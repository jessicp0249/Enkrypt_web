Create coded messages

How to use

Type a message into the text box below and click the "EnKrypt!" button to encode it.
Click the "Clear" button to clear the text box. Refresh this page to clear the coded output.
Use the Display options to change the size and colors of the coded output.

    For best results, use only the following:
  • Letters A-Z (lowercase or uppercase)
  • Numbers 0-9
  • Periods, exclamation points, question marks (. ! ?)
  • Single and double quotes (' ")
